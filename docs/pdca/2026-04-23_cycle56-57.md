## 判定

### ✅ Cross-asset generalization 確立

**3 asset × 4 期間 = 12 マス全てで v6 >= v5**。

特に注目点:
1. **BTC 2y で v5 -0.36% → v6 +0.25% とマイナス脱出** — allPositive を新規に達成
2. **ETH 2y で +2.77pp** (LTC の +2.15pp と同規模)
3. **BTC aggregate geomMean は 8.6倍改善** (+0.03% → +0.25%)

→ **v6 の改善 (bb_squeeze_lookback=3 + cmf_buy_min=0.10) は LTC 特有ではなく 15m 系で汎化する改善**。単一 asset の curve-fit ではない。

### 必須制約チェック

- LTC/BTC/ETH すべて **MaxDD ≤ 20%** (最大 6.86%, ほぼすべて 5% 以下)
- **すべて allPositive** (BTC v6 含む — v5 は false だった)

## 学び

1. **Profile 仮説の asset-agnostic 性**: bb_squeeze (ボリンジャーバンドの squeeze 判定) と CMF (資金流入) は crypto 全般の price action に共通するシグナルなので、LTC で効いた improvement が他通貨でも効くのは技術的に自然。cycle28-37 の fragility は **asset-specific な 2022 LTC 暴落への curve-fit** であり、v6 の improvement は asset-general な改善であることが対比的に示された。
2. **agent-guide §1 "LTC/JPY 固定" の再解釈**: 制約は **運用対象 = LTC/JPY** であり、**実験で cross-asset 検証** は範囲内。今後 BTC/ETH で追試しておく運用が v6 以降の promote で常套化する価値あり。
3. **BTC は 15m では低 ratio**: 全期間 return が 1% 未満 — BTC は本来長周期でトレードする asset なので 15m では signal が希薄。しかし **v6 の improvement 比率は BTC で最大** (x8.6)。これは BTC の scalp-friendly に v6 のフィルタが合っている可能性。
