## 判定

### H1 (4期間評価) — ✅ pass

production は 3m / 6m / 1y / 2y **すべてで正のリターン (min 0.95%, max 12.88%)**、MaxDD **すべて 20% 以下 (min 1.37%, max 6.86%)**、aggregate geomMean +6.88%。LTC/JPY 15m の現行 v5 sl14 戦略は **直近 2 年までの範囲で必須制約を満たし運用可能**と確認。

ただし **2022 regime (2022-10-28 以降を含む 4y) では −48%, DD 91% で破綻** — これは cycle28-37 の既知 fragility で本 cycle でも解消しない。**直近 2 年の範囲という user 指示の制約下では問題ない** が、運用期間を 2022 まで延ばす場合は defensive profile (`experiment_2026-04-22_sl6_tr30_tp6_tf60_35`) への切替か regime detection が必要。

### H2 (multi-axis compound) — ✅ **partial promote 候補**

**combined (bblk3 + cmf010) の特徴**:
- **2y で baseline +12.88% → +15.03% (+2.15pp, +16.7% 相対改善)** — 最大の利得
- **2y MaxDD 6.86% → 6.54% (-0.32pp)** — DD 同時改善
- **aggregate geomMean +6.88% → +7.32% (+0.44pp)** — bblk3 単独 (+0.18pp) + cmf010 単独 (+0.24pp) の和とほぼ一致し、**軸は独立**
- 1y / 2y / aggregate で baseline を超えるが、**3m / 6m でわずかに劣化 (-0.15pp / -0.22pp)** — 短期では微減
- 3y (2023-04〜2026-04) でも +1.47pp と改善持続

**promote 推奨度**: 中。長期 (1y+ / 2y+) では明確に上回るが、直近 3m / 6m で劣化するため **即時 production 上書きは保留**。次 cycle で trailing ATR multiplier や trend_follow rsi_buy_max などの直交軸と組合せて短期 regime のカバーを狙う価値あり。

### H3 (WFO robustness) — 🟡 **partial pass**

- `cmf_buy_min=0.1` は 9/10 窓で勝者 → **robust (promote 候補)**
- `bb_squeeze_lookback` は値選択が regime 依存で単独 robust ではない (3:4, 10:4, 5:1)
- OOS aggregate `allPositive=false` だが worst は −2.60% で深刻な負ではなく、2023 後半〜2024 の既知弱 regime を反映したもの (baseline でも同様に負)

### 必須制約チェック

- ✅ **MaxDrawdown ≤ 20%** (4 期間すべて、max 6.86%)
- ✅ **TotalReturn 正** (4 期間すべて)
- 🟡 **BiweeklyWinRate** 目標 80 に対して最高 61 (現行戦略全体の体質)
- ✅ **allPositive (4 period aggregate)** — production / combined 共に true

## 学び

1. **multi-axis の独立性が確認できた**: 単独効果 (bblk3 +0.18pp, cmf010 +0.24pp) がほぼ線形に足し合わさって combined +0.44pp になっており、2 軸は BREAKOUT adjacent でも別メカニズムで効いている。今後も `bb_squeeze_lookback` × `cmf_buy_min` は compound しやすいペアとして扱える。
2. **短期劣化のリスク**: combined は 1y/2y に最適化されるほど 3m/6m が下がる傾向。これは PDCA README の「最低 1 ヶ月、推奨 3〜6 ヶ月」と整合し、**短期は参考値**として扱うべき。ただし user が直近評価を重視するなら 3m の劣化幅 (-0.15pp, absolute -0.0024%) は許容範囲。
3. **WFO robustness で cmf_buy_min=0.1 が確立**: 9/10 勝率は PDCA README の「≥ 6/10 窓で robust」基準を大きく上回る。**cmf010 単独の v6 minor bump 候補として独立に検討する価値がある** — 3m/6m/1y/2y すべてで baseline 同等以上を確認済みで副作用が小さい。
4. **2022 regime fragility は multi-axis でも解消しない**: combined でも 4y incl 2022 は −46.67% (+1.64pp だけ) で、別 PR (regime routing 再活性化 / asset diversification) が必要。今回の cycle45 範囲外。

## 次サイクル提案

- ~~**A 案: cmf010 単独 v6 promote**~~ — 単軸 WFO が 3/10 で insufficient、**撤回**。
- **B 案 (最有力): combined promote + 短期補強**: combined を base に、3m/6m 劣化を補うために `signal_rules.trend_follow.rsi_buy_max` や `trailing_atr_multiplier` を追加 axis に sweep。combined は 2y / 3y / aggregate geomMean で明確に勝ち、WFO 10 windows aggregate は negative windows は baseline と同一なので純粋な追加は upside のみ。
